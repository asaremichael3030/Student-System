# 📚 Student Management System

The Student Management System is a web-based application designed to help manage student data efficiently. It allows users to add, update, view, and delete student records in a simple and user-friendly interface.
This project is ideal for schools, administrators, or developers learning how to build CRUD (Create, Read, Update, Delete) applications.

 ✨ Features
* ➕ Add new students
* 📋 View all student records
* ✏️ Edit student information
* ❌ Delete student records
* 🔍 Search and filter students (optional)
* 📱 Responsive design for desktop and mobile
